## Description

Backend часть проекта автоматизации очереди доступа к паркингу.

[Документация проекта](https://parking-project-docs.notion.site/c1c706fd717641cab58a2a76a06dbadf?v=34372ff7fdf54053955e45f7f9758804)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit test
$ npm run test

# e2e test
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
